Documentation

Concept

The goal of our project is to create a real time web API for angle of arrival and range information of a tag.

We would like to use two, fixed ultra wide band (UWB) radio transceivers (we'll call these Anchors) to collect angle of arrival and range information of a third, mobile UWB transceiver (we'll call this the Tag).

Motivation

Our motivation for this project is to take the new UWB technology and create an open source, cheap, and accessable way to use it in order to get very precise angle of arrival and range information in a repeatabe way.

Competitive Analysis

Because there is no publicly available solution to our problem, we don't have any direct competators. However, we thought this would be a good time to mention competators to the UWB technology which will help explain why we are chosing to use it.

Wifi-based

Redpin is an open source indoor positioning system that was developed with the goal of providing at least room level accuracy. This technology can be used on any wifi-connected device.

Bluetooth Low Energy (BLE) Beacons

Estimote is a company that creates small, wireless BLE radios that can be attached to any location or object in order to track surrounding smartphones. Estimote also has an SDK on top of which apps can be written in order to interact with their radios. Estimote boasts an accuracy of 1 to 4 meters with measurement error of 20-30% of actual distance.

Ultra Wide Band (UWB)

UWB radios use time of flight (TOF) analysis to gain distance readings with sub-meter accuracy. Currently, UWB is often used in custom installations because the biggest player in the space Decawave is a closed source shop and charges large amounts to gain access to their localization-enabled kits. There is no open SDK for interfacing with the Decawave radios.
